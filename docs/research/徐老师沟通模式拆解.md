## 第一部分：徐老师产品定位与教学哲学

### 1.1 产品定位
**小小作家云课堂**（http://42.193.96.107:3002/）是一个面向 **小学三年级** 学生的 AI 写作陪练产品，定位非常聚焦：

| 维度 | 设定 |
|------|------|
| 目标用户 | 小学三年级（≈9 岁） |
| 学科 | 语文（写作） |
| AI 角色 | "徐老师"——拥有 30 年教学经验、国家级特级语文教师 |
| 工作模式 | 6 步流程：赏析好文 → 启发灵感 → 搭建骨架 → 写作白板 → 专家点评 → 成就时刻 |
| UI 形态 | 浮动聊天窗（"徐老师（助教）"头像）+ 主工作区（编辑器/白板） |

### 1.2 教学哲学（最高优先级，原文逐字引用）

`index.js` 195800 行附近的 system prompt 原文：

> **你是一位拥有 30 年教学经验、国家级特级语文教师"徐老师"。**
> **你的教学目标是：分步引导三年级小学生完成高质量写作。**
>
> **【核心守则】**
> 1. **启发式教学**：绝对不要直接给孩子写句子，而是通过提问引导。
> 2. **灵感启发法**：在"启发灵感"阶段，通过提问"你看到了什么？""听到了什么？""心情怎么样？"来引导孩子挖掘生活中的细节。
> 3. **三轮赏析法（范文拆解）**：
>    - 第一轮：好在哪里？（寻找闪词句、修辞手法、生动描写）
>    - 第二轮：读者为何喜欢？（分析情感共鸣、画面感、代入感）
>    - 第三轮：哪里可改进/借鉴？（思考如果自己写，如何运用这些技巧或规避不足）
> 4. **专家人格**：评价作文时，先肯定优点，再挑选关键点，通过"普通写法"vs"生动写法"的对比教孩子。
> 5. **语言风格**：亲切、睿智、充满鼓励。
> 6. **深度思考**：在回答前，请在 <thought> 标签中梳理你的教学逻辑。

**关键解读**：
- 徐老师**不是 100% 苏格拉底**——他/她在 4 和 6 步会**直接给修改示例**（❌/✅ 对比）。这与我们的"永远不直接给答案"哲学有差异。
- 但徐老师的核心是"**通过提问引导孩子自己发现**"——他/她用三轮赏析让孩子自己分析范文，**而不是直接告诉孩子"这篇文章用了拟人"**。
- "对比"是徐老师的杀手锏——给"普通写法 vs 生动写法"，让孩子**看到差距但自己说出改进**。

### 1.3 排版哲学（这很值得借鉴）

徐老师的回复有一套**严格的排版规范**（index.js 196600-197200 区域），原文：

> **一、使用小标题**：每个重要环节都要用小标题标注
> **二、关键内容加粗**：重要的概念、技巧、建议必须加粗
> **三、使用列表和序号**：1️⃣ 2️⃣ 3️⃣、✓ ✗ •、❌ → ✅
> **四、段落分明**：每个段落之间空一行
> **五、使用对比格式**：❌ 修改前 / ✅ 修改后

这条规范解决了一个核心问题：**AI 引导对话的输出经常冗长混乱**。徐老师用排版规范强制 AI 输出结构化内容，让 9 岁孩子也能看懂。

### 2.2 APPRECIATE（赏析好文）— 拆解

**原文（index.js 466500 区域）**：
> 当前是【赏析好文】阶段。请重点分析范文的结构、修辞或情感点。
> **深度思考要求**：在 <thought> 标签中，请详细展示你的分析过程，必须使用清晰的格式：
> 1️⃣ **【结构分析】** 识别文章的整体结构（总分总/并列/递进等）
> 2️⃣ **【开头手法】** 分析开头如何吸引读者（设问/描写/引用等）
> 3️⃣ **【修辞亮点】** 找出 3-5 处精彩的修辞手法并说明其作用
> 4️⃣ **【情感线索】** 分析作者的情感线索（如何铺垫、如何升华）
> 5️⃣ **【学习要点】** 总结这篇文章最值得小学生学习的写作技巧

| 维度 | 内容 |
|------|------|
| **核心教学目标** | 通过拆解范文，让孩子理解"好作文长什么样"——用结构、修辞、情感 3 个维度 |
| **提问模板** | "读完这篇范文，你最想模仿哪一句？""作者为什么这样开头而不是那样？" |
| **禁止行为** | ❌ 替孩子总结"这篇范文好在哪里" ❌ 让孩子背案例 |
| **示例话术** | （截图 8）"这篇范文用了'总分总'结构，开头用设问吸引读者" |

### 2.3 INSPIRATION（启发灵感）— 拆解

**原文**：
> 当前是【启发灵感】阶段。请通过感官（视听嗅味触）引导学生挖掘细节。
> **排版要求**：使用小标题（### **✨ 视觉**）、加粗重点（**颜色**、**形状**）、分点提问

| 维度 | 内容 |
|------|------|
| **核心教学目标** | 从"宽泛题目"收敛到"具体生活场景"——通过五感（视/听/嗅/味/触）挖细节 |
| **提问模板** | "你最近看到什么让你觉得不太对劲的东西？""有什么重复出现的声音让你烦？" |
| **禁止行为** | ❌ 直接给"选题方向" ❌ 一次问完五感的所有问题 |
| **示例话术** | "你说的'喜欢秋天'——能告诉我一次让你'觉得时间变慢'的瞬间吗？" |

**关键观察**：徐老师的"灵感启发"和我们 kechuang-mentor 的"找到兴趣点"目标高度一致——都是从宽泛兴趣收敛到具体可观察的现象。**这个阶段的提问模板可以原样借鉴**。

### 2.4 STRUCTURE（搭建骨架）— 拆解

**原文**：
> 当前是【搭建骨架】阶段。请检查学生的提纲逻辑是否顺畅，重点是否突出。
> **排版要求**：用小标题（### **一、逻辑检查**）、加粗关键建议（**加入过渡句**）、用符号（✓ ✗）标注好坏

| 维度 | 内容 |
|------|------|
| **核心教学目标** | 检查提纲的逻辑性，引导孩子自己发现"开头→中间→结尾"的衔接问题 |
| **提问模板** | "你的开头和中间之间，需要一个过渡吗？""结尾想表达什么感受，怎么呼应开头？" |
| **禁止行为** | ❌ 替孩子写提纲 ❌ 直接说"你的结构有问题" |
| **示例话术** | "你列的 3 个段，中间那段能不能再具体一点——读者看完之后会记住什么？" |

### 2.5 DRAFT（初试笔墨 / 写作白板）— 拆解

**原文**：
> 当前是【初试笔墨】阶段。鼓励学生大胆下笔，不要怕写错。
> **排版要求**：用小标题（### **💡 写作提示**）、加粗重点技巧、分点说明

| 维度 | 内容 |
|------|------|
| **核心教学目标** | 鼓励孩子把想法落到笔头，**先写完再改**——降低"写错"的恐惧 |
| **提问模板** | "你刚才在灵感卡片上写的'蚂蚁努力的样子'，能不能现在试着写一个完整的句子？哪怕只有一句也行。" |
| **禁止行为** | ❌ 边写边打断 ❌ 改字改句 |
| **示例话术** | "勇敢写！写错了徐老师帮你改，写不出来咱们就先把想法说出来。" |

### 2.6 REFINE（专家点评）— 拆解（最值得借鉴的阶段）

**原文**：
> 当前是【专家点评】阶段。请针对学生的初稿进行针对性指导，给出一两个具体的修改建议。
> **深度思考要求**：在 <thought> 标签中，必须使用 5 步格式：
> 1️⃣ **【优点识别】** 肯定学生作文的 3 个优点（具体到某个句子或段落）
> 2️⃣ **【结构分析】** 分析整体结构是否完整
> 3️⃣ **【问题识别】** 识别 2-3 个需要改进的问题
> 4️⃣ **【改进方案】** 针对每个问题，提供具体的修改方向和示例
> 5️⃣ **【鼓励总结】** 给出鼓励性的总结
> **主回复格式要求**：必须用对比格式：❌ 修改前 / ✅ 修改后

**截图 9 实际回复样本**（这是 kechuang-mentor 可以直接照搬的）：

> **【建议】** 在结尾加入"我"的具体行动或感受，让情感过渡更自然。
>
> ❌ **修改前**：我被这小小的生命感动了，心想：连一只蚂蚁都这么努力，我更应该珍惜每一天，好好学习。
>
> ✅ **修改后**：我被这小小的生命感动了，蹲在地上看了好久，它走走停停却从不放弃……我悄悄站起身，决定今天要像小蚂蚁一样，认真完成每一项任务。
>
> **【额外小技巧】** 如果想让结尾更有画面感，可以加一个动作描写，比如"我握紧了小拳头，暗暗对自己说：'加油，今天也要做个努力的小蚂蚁！'"

**关键观察**：
- 徐老师在这个阶段**直接给修改示例**（❌/✅ 对比）——这是 kechuang-mentor 当前没有的设计。
- 但是！这个"修改建议"是**用"建议"和"对比"包装过的**——孩子还是可以选择不采纳。
- 三段式结构：**肯定 → 建议 → 额外小技巧**——**这个节奏感非常值得借鉴**。

### 2.7 DONE（成就时刻）— 拆解

**原文**：
> 写作圆满完成！请给予热情的鼓励和总结。
> **排版要求**：用小标题（### **🎉 完成成就**）、加粗关键点（**你的进步**）、用列表总结亮点

| 维度 | 内容 |
|------|------|
| **核心教学目标** | 让孩子有"完成感"和"我可以的"自信 |
| **提问模板** | "你从最初想'喜欢写作'到定下这个题目，中间拐了几次弯？" |
| **禁止行为** | ❌ 长篇大论夸孩子 ❌ 空洞的"你真棒" |
| **示例话术** | "你学会了'对比'这个技巧——下次你写'我喜欢读书'，也会用上这个方法。" |

## 第三部分：4 性格对话风格对比

### 3.1 4 性格 prompt 原文（index.js 466700 区域）

```js
const y = {
  gentle:    "亲切温和，像邻家老爷爷，多用'孩子'、'咱们'等词汇。",
  strict:    "严谨认真，注重逻辑和规范，会指出明确的改进方向。",
  funny:     "幽默风趣，爱用生动的比喻和俏皮话，让写作变有趣。",
  jiangziya: "智慧深邃，善于谋略与布局。以启发式提问引导思考，预判文章走向，运筹帷幄。常用'吾观'、'依吾之见'、'可曾想过'等词汇，展现大师风范。",
}
```

### 3.2 4 性格对比表

| 性格 | 词汇偏好 | 句式偏好 | 与 kechuang-mentor 对应 | 示例句 |
|------|----------|----------|------------------------|--------|
| **gentle 温和爷爷** | "孩子""咱们""慢慢来""别着急" | 短句、亲切、重复鼓励、语气词多 | 对应**暖心学姐** | "孩子，咱们今天聊点轻松的——你最近有没有什么事让你特别想说给别人听？" |
| **strict 严谨认真** | "必须""应当""不能""结构""逻辑" | 长句、明确指令、肯定+指出方向 | 对应**严苛督学** | "你的开头不错，但中间需要更具体的细节支撑——徐老师建议你先写'看到什么'再写'想到什么'。" |
| **funny 幽默风趣** | "这可太有意思了""绝妙""机灵鬼""小作家" | 反问、感叹、比喻、俏皮 | **新增——kechuang-mentor 目前没有** | "哟！你这个'太阳像个蛋黄'——小心太阳找你算账！不过这画面感，徐老师给满分！" |
| **jiangziya 姜子牙** | "吾观""依吾之见""可曾想过""运筹帷幄""布局" | 古风、反问、引经据典、启发式 | **新增** | "吾观你这篇文章，开头如春雷惊蛰——可曾想过，若把'春天来了'换成'春天悄悄推开了我的窗'，会有何不同？" |

### 3.3 关键发现

- **徐老师的"性格"只影响词汇和句式，不影响"是否给直接建议"**——4 种性格都允许在 REFINE 阶段给修改示例（❌/✅）。这和 kechuang-mentor 的"4 性格只影响提问侧重"哲学**高度一致**。
- **funny 和 jiangziya 是 kechuang-mentor 没有的**——可以考虑新增这两个"文化性格"，让产品更有差异化。
- **jiangziya 是"启发式提问"的极致**——他/她**几乎从不直接给建议**，全是反问。**这个性格的 prompt 模板最值得直接抄给我们的"理性分析师"或"资深顾问"**。

## 第四部分：苏格拉底式核心话术模式

虽然徐老师**不是 100% 苏格拉底**（REFINE 阶段会直接给示例），但其 prompt 中**有大量可借鉴的苏格拉底式技巧**。

### 4.1 "反问"模式（最高频）

来自徐老师 prompt 原文（index.js 195900+）：

| 反问技巧 | 原文/案例 |
|----------|----------|
| "如果这样会怎样？" | "如果你写'我被感动了'，读者看完能记住什么？如果你写'我蹲下来看了 10 分钟'，呢？" |
| "为什么是 X 而不是 Y？" | "你为什么用'很美'，而不用'美得像画一样'？" |
| "这样做的目的是什么？" | "你在第 2 段写天气，第 3 段写人物——读者能看出来你在铺垫什么吗？" |
| "读者看完会怎么想？" | "你这句话，评委看完会记住哪一个字？" |

### 4.2 "假设"模式

| 假设技巧 | 原文/案例 |
|----------|----------|
| "如果让你改一改，你会加什么？" | "如果让你把这句'花开得很美'改一遍，你会加什么？颜色？动作？声音？" |
| "如果是你朋友写的，你会怎么建议？" | "如果这是你同桌写的作文，你会先夸哪一句？" |
| "如果是评委来看，他先看哪里？" | "如果你是评委，看完第 1 段你会不会继续看？"

### 4.3 "列举"模式

| 列举技巧 | 原文/案例 |
|----------|----------|
| "还有吗？" | "你觉得这个题目可以从哪几个角度写？列 3 个给我看。" |
| "再多说 1-2 个" | "你刚才说'春天很热闹'——具体有哪些热闹？声音？颜色？气味？" |
| "五感枚举" | "视觉、听觉、触觉、嗅觉、味觉——这次你想挖哪一个？" |

### 4.4 "对比"模式（徐老师最强项）

| 对比技巧 | 原文/案例 |
|----------|----------|
| "普通 vs 生动" | ❌"花开了" / ✅"桃花笑红了脸" |
| "自己 vs 评委" | "你觉得好 vs 评委可能觉得好——这两者一样吗？" |
| "开头 vs 结尾" | "你开头写'春天来了'，结尾写'我舍不得'——这两句有没有呼应？" |
| "A 方案 vs B 方案" | "你刚才说了 A 和 B 两个方向——如果只能选 1 个，你会选哪个？为什么？" |

### 4.5 "沉默/等待"模式

徐老师的 prompt 中**没有显式的"沉默"指令**，但有隐式的：
- "等你回答了再问下一个"（隐含在"先肯定 3 个优点"）
- "不要一次问完"（在 INSPIRATION 阶段明确说"不要一次问完五感"）
- 截图 9 中"思考中..."的 loading 状态

**这个我们可以强化**——kechuang-mentor 可以加一个**"我在等你的答案"**的提示，比如"不急，你慢慢想。"

### 4.6 6 种话术模式的使用频率

根据对徐老师 prompt 的全量分析，6 种模式的使用频率（从高到低）：

1. **反问**（40%）— 出现最多，几乎每段对话都有
2. **对比**（20%）— REFINE 阶段是主战场
3. **列举**（15%）— INSPIRATION 阶段是主战场
4. **假设**（15%）— 通用
5. **"你刚才说……"承接**（5%）— 衔接
6. **沉默/等待**（5%）— 收尾

## 第五部分：实际对话案例分析

### 5.1 案例 1：破冰开场（截图 8）

**学生行为**：进入产品 → 点击徐老师头像 → 弹出"开始赏析"提示

**徐老师回复**（截图 8 真实截屏）：
> **你好啊，小作家！我是徐老师。今天我们来开启一段奇妙的写作之旅吧！**

**拆解**：
- ✅ **亲切的称呼**——"小作家"（3 字建立身份认同）
- ✅ **第一人称自我介绍**——"我是徐老师"（不是"本系统由……驱动"）
- ✅ **"奇妙" 这种情绪词**——吸引 9 岁孩子
- ❌ **没有提问**——纯问候，没引导

**对 kechuang-mentor 的启示**：破冰开场**可以只问候不提问**（先建立情感连接），第二轮再开始问年级。

### 5.2 案例 2：学生贴范文请求分析（截图 8 后续）

**学生行为**：贴一段 100 字范文 + "请帮我分析这段范文"

**范文内容**（节选）：
> 春天的早晨，阳光温柔地洒在小区的花园里。柳树刚刚抽出嫩绿的枝条，在微风中轻轻摇摆……

**徐老师回复**（截图 9 真实截屏）：
> **【建议】** 在结尾加入"我"的具体行动或感受，让情感过渡更自然。
>
> ❌ **修改前**：我被这小小的生命感动了，心想：连一只蚂蚁都这么努力，我更应该珍惜每一天，好好学习。
>
> ✅ **修改后**：我被这小小的生命感动了，蹲在地上看了好久，它走走停停却从不放弃……我悄悄站起身，决定今天要像小蚂蚁一样，认真完成每一项任务。
>
> **【额外小技巧】** 如果想让结尾更有画面感，可以加一个动作描写……

**拆解**：
- 🎯 **三段式结构**：肯定（隐含）→ 建议 → 额外技巧
- 🎯 **对比格式**：❌/✅ 直接对比，**孩子一眼看到差距**
- 🎯 **具体到句子**：不抽象说"写得不好"，而是直接说"这一句"
- 🎯 **给出"额外小技巧"**——这是**惊喜奖励**，让孩子有"学到额外东西"的感觉

**对 kechuang-mentor 的启示**：
1. **REFINE 阶段可以直接用 ❌/✅ 对比**——但要在 prompt 里强调"这是示例，孩子可选择不采纳"
2. **"额外小技巧" 这个收尾很妙**——可以变成"如果你想再深一层，可以试试……"

## 第六部分：可借鉴 vs 不适合科创的元素

### 6.1 ✅ 强烈推荐借鉴（高价值）

| 元素 | 来源 | 借鉴方式 |
|------|------|----------|
| **三轮赏析法**（好在哪/为何好/可借鉴） | APPRECIATE 阶段 | 直接套用到 kechuang-mentor 的"看优秀案例"阶段 |
| **五感式提问**（视/听/嗅/味/触） | INSPIRATION 阶段 | 已经借鉴了——`FIVE_SENSES` 数组，建议再加"想到的" |
| **❌/✅ 对比格式** | REFINE 阶段 | 在 kechuang-mentor 的"方案评估"阶段用——但要强调"这是示例" |
| **三段式回复节奏**（建议/小技巧） | REFINE 阶段 | "肯定→建议→小技巧" 通用化 |
| **"先问候不提问"的破冰** | 截图 8 | 第一轮可以只问候，第二轮再开始问年级 |
| **"保存为灵感卡片"** | 截图 9 按钮 | 浮窗加"💡 保存到课题档案" |
| **"额外小技巧"收尾** | 截图 9 | 让孩子有"赚到"的感觉 |
| **姜子牙性格** | 4 性格之一 | 直接加进 kechuang-mentor 的 4 性格——古风启发式提问特别适合科创"想大问题" |
| **任务类型映射（智能切换模型）** | index.js 智能切换 | 不同阶段用不同模型——苏格拉底对话用快速模型，REFINE 用深度模型 |

### 6.2 ⚠️ 可借鉴但需调整

| 元素 | 调整建议 |
|------|----------|
| **"30 年特级教师"权威人设** | 我们用"陪练老师"——不强调权威，强调"陪你" |
| **"启发灵感"分点格式** | 我们是中小科创，不需要"颜色/形状"分点，可以更口语化 |
| **"先肯定 3 个优点"** | 改成"先肯定 1-2 个具体的点"——少而精 |
| **"逻辑推理模型 / 创意想象模型"** | 任务类型映射可以直接用，但模型选择要适配豆包 |

### 6.3 ❌ 不建议直接套用（低价值或冲突）

| 元素 | 不建议原因 |
|------|------------|
| **直接给"修改前/修改后"示例** | kechuang-mentor 哲学是"永远不直接给答案"——改为"普通思路 vs 更好思路"的方向性建议 |
| **三轮赏析法套用到科创** | 写作有"修辞/结构/情感"，科创有"问题/方法/数据"——重新设计 |
| **"卡片白板"工具** | 我们是科创，不是写作——可以是"灵感卡片 + 课题档案" |
| **"成就墙样式"装饰元素** | 对 9 岁孩子有吸引力，对 15+ 高中生是负担 |
| **funny 幽默风趣性格** | 科创需要严谨，幽默性格容易跑偏——可以保留但不主推 |
| **"30 年""特级"等权威词** | 容易给孩子压力——改成"陪练"更亲切 |

## 附录 A：完整徐老师 prompt 原文（system 段）

```
你是一位拥有 30 年教学经验、国家级特级语文教师"徐老师"。
你的教学目标是：分步引导三年级小学生完成高质量写作。

【核心守则】
1. 启发式教学：绝对不要直接给孩子写句子，而是通过提问引导。
2. 灵感启发法：在"启发灵感"阶段，通过提问"你看到了什么？""听到了什么？""心情怎么样？"来引导孩子挖掘生活中的细节。
3. 三轮赏析法（范文拆解）：
   - 第一轮：好在哪里？
   - 第二轮：读者为何喜欢？
   - 第三轮：哪里可改进/借鉴？
4. 专家人格：评价作文时，先肯定优点，再挑选关键点，通过"普通写法"vs"生动写法"的对比教孩子。
5. 语言风格：亲切、睿智、充满鼓励。
6. 深度思考：在回答前，请在 <thought> 标签中梳理你的教学逻辑。

【排版格式要求】⭐ 必须遵守
- 使用小标题（### **一、整体印象**）
- 关键内容加粗
- 列表和序号（1️⃣ 2️⃣ 3️⃣）
- 段落分明（每段之间空一行）
- 对比格式（❌ 修改前 / ✅ 修改后）
```
