# 科创导师 · 开发路线图

---

## 阶段 0:契约对齐 ✅

- 读 PROJECT_BRIEF.md 和 MULTI_AGENT_PLAN.md
- 对齐苏格拉底哲学、4 性格、3 段年级、3 比赛 JSON

## 阶段 1:项目骨架 ✅

- Vite + React 19 + Tailwind 初始化
- 暗金/深蓝主题
- 目录结构(components/data/prompts/server)

## 阶段 2:数据层 ✅

- `data/cases.js` — 5 个真实获奖课题
- `data/competitions.js` — 3 比赛 + 路由逻辑
- `data/personalities.js` — 4 性格(苏格拉底底色)
- `data/age_adaptations.js` — 3 段年级适配

## 阶段 3:Prompt 层 ✅

- `prompts/system.js` — 苏格拉底教学哲学(最高优先级)
- `prompts/onboarding.js` — 破冰对话
- `prompts/appreciate.js` — 看优秀案例
- `prompts/inspiration.js` — 找到兴趣点
- `prompts/structure.js` — 课题方向聚焦
- `prompts/draft.js` — 研究方案设计
- `prompts/refine.js` — 方案评估(不暴露评分)
- `prompts/achievement.js` — 项目档案
- `prompts/index.js` — 统一出口

## 阶段 4:前端组件 ✅

- `StepNav` — 6 步顶部导航
- `Sidebar` — 左侧栏(档案 + AI 设置 + 素材)
- `AIAssistant` — 浮动 AI 引导窗(可拖动)
- `AISettings` — 性格/模型切换
- `ExportButton` — 4 格式导出
- `StudentProfile` — 破冰对话
- 6 个 step 组件

## 阶段 5:后端 + 评估路由 ✅

- `server/index.js` — Express 入口
- `server/routes/health.js` — 健康检查
- `server/routes/ai.js` — 豆包对话代理
- `server/routes/evaluate.js` — 评估引擎(注入 3 比赛 JSON)
- `server/services/volcengine.js` — 火山引擎代理 + mock 降级

## 阶段 6:文档 + 示例 ✅

- `README.md` — 启动 + GitHub push + 决策记录
- `docs/产品PRD.md` — 简化版 PRD
- `docs/技术架构.md` — 前后端架构 + 数据流
- `docs/开发路线图.md` — 本文件
- `examples/sample_topic.json` — 示例课题
