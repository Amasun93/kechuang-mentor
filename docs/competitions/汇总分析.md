> **目标用户**：小学生（5-6年级）/ 初中生（12-15岁）/ 高中生（3档用户）
> **数据来源**：3 个比赛官网/官方通知、地方政府红头文件、中国青年报/人民网/新华网等权威媒体、CSDN/知乎获奖经验分享、青少年活动平台公布的获奖名单

> **覆盖比赛**：① 全国青少年科技创新大赛（青创赛）② 宋庆龄少年儿童发明奖 ③ 雏鹰杯红领巾科创达人挑战赛

## 二、3 个比赛评审维度的「共性」与「差异」

### 共性 6 大维度（3 个比赛都看重）

尽管 3 个比赛定位不同，但评委脑里的「核心关切」高度一致。可以提炼为 6 大共性维度：

| 共性维度 | 核心问题 | 出现频次 |
|---|---|---|
| ① **创新性** | 这件事别人做过吗？跟已有方案有什么本质不同？ | 3/3 比赛均列为最重要维度 |
| ② **实用性/真实问题** | 解决了什么真实问题？目标用户是谁？方案能用吗？ | 3/3 比赛均赋予高权重 |
| ③ **科学性/研究规范** | 研究方法科学吗？数据真实吗？记录完整吗？ | 3/3 比赛均要求 |
| ④ **完整性/工作量** | 研究链条完整吗？过程记录齐全吗？工作饱满吗？ | 3/3 比赛均考察 |
| ⑤ **学生主体性** | 这真的是学生做的吗？学生能讲清楚每个细节吗？ | 3/3 比赛均作为底线（造假即淘汰） |
| ⑥ **现场答辩与展示** | 答辩逻辑清晰吗？展板/演示能让人信服吗？ | 3/3 比赛均设现场环节 |

### 差异：3 个比赛的「个性」侧重

| 差异点 | 青创赛 | 宋庆龄 | 雏鹰杯 |
|---|---|---|---|
| **最看重什么** | 科研潜质（要求立论严谨、论据充分） | 新颖性+实用性（必须有实物/可演示） | 真实问题+科学方法（从课堂走向真实世界） |
| **作品形态** | A类（专业实验室）vs B类（日常生活） | 发明/创意/AI编程/科技绘画 4 类 | 7 大方向（创造发明/自然生态/智慧城市/航空航天/AI/新能源车/创新创意） |
| **是否要求实物** | 鼓励但不强制 | **发明作品必须有实物** | **创造发明类必须有实物或仿真模型** |
| **是否接受论文** | ✅ 是（中学生可写论文） | ❌ 明确**不接受**科学研究论文类 | ❌ 偏向实物/程序/视频 |
| **A/B类分类** | ✅ 按专业程度分 A/B 类 | ❌ 不分 | ❌ 不分，但 7 大方向分类 |
| **是否限制主题** | 自由选题 | 自由选题（不限主题） | 每年有年度主题（2025：「智慧关怀、智学未来、智行上海」） |
| **特色加分项** | 与学科分类深度结合（数学/物理/化学/生命科学/计算机/工程/环境/行为社科） | 绿色环保/可持续发展 | 「智慧关怀」「智学未来」「智行上海」专项 |
| **对学术规范要求** | 极高（查重≤30%，查新报告，原始数据） | 中等（强调查新） | 较低（重在真实参与） |
| **评审团规格** | 全国学科专家 | 中国发明协会+基金会专家 | **两院院士**领衔 |
| **奖项含金量** | 国家级一等奖是升学硬通货 | 国家级奖项 | 「小院士」称号在上海升学中含金量高 |

## 三、3 个比赛评审维度的「共性」与「差异」

### 3.1 一句话定位

- **青创赛** = 「**科研训练场**」——比的是研究的规范性和科学素养
- **宋庆龄** = 「**发明创造展**」——比的是把脑子里的想法做成实物
- **雏鹰杯** = 「**真问题实验室**」——比的是用科学方法解决真实问题

### 3.2 评估侧重差异（按权重从高到低）

| 排名 | 青创赛 | 宋庆龄 | 雏鹰杯 |
|---|---|---|---|
| 1 | 创新性 25% | 新颖性 30% | 创新性 25% |
| 2 | 科学性 20% | 实用性 25% | 实用性 25% |
| 3 | 完整性 15% | 完整性 15% | 科学性 20% |
| 4 | 实用性 15% | 创造性 10% | 学生主体性 15% |
| 5 | 选题适配 10% | 参与度 10% | 作品形态 10% |
| 6 | 研究过程 10% | 现场表现 5% | 现场答辩 5% |
| 7 | 现场表现 5% | 绿色环保 5% | — |

**洞察**：
- 3 个比赛都把「创新性+实用性」合计放在 **40-55%** 的最高权重区
- 青创赛和宋庆龄的差异主要在「**学术规范**」vs「**实物作品**」——前者重研究过程严谨，后者重作品本身的新颖
- 雏鹰杯的特点是「**学生主体性**」单独占 15%，比另外两个比赛更看重（5-10%）——因为院士级评委特别强调「真问题、真研究、真参与」

## 四、年龄段适配性对比（核心新增章节）

### 4.1 3 个比赛对 3 档用户的「适配度」总览

| 学段 | 青创赛 | 宋庆龄发明奖 | 雏鹰杯 | 推荐排序 |
|---|---|---|---|---|
| **小学中低年级（1-3年级）** | ⚠️ 不推荐（认知能力不足；只能报B类且多数为低分） | ✅✅ 非常推荐（科技绘画类专为小学生设；发明/创意类对低年级也开放） | ⚠️ 不推荐（认知能力不足；最低2年级勉强可报「创新创意」类） | **宋庆龄 > 雏鹰杯 > 青创赛** |
| **小学高年级（4-6年级）** | ✅ 推荐（只能报B类，但拿奖机会大；很多家长把这里作为升学跳板） | ✅✅ 非常推荐（4类均可报；与初中阶段作品类别有衔接） | ✅✅ 非常推荐（7大类均可报；「小院士」称号含金量高） | **雏鹰杯 ≈ 宋庆龄 > 青创赛** |
| **初中（7-9年级）** | ✅✅ 非常推荐（产品主要目标用户；B类为主，也可冲A类） | ✅✅ 非常推荐（是产出金奖的主力军） | ✅✅ 非常推荐（核心组别；院士评委最看重） | **3 个比赛都重点推荐** |
| **高中（10-12年级）** | ✅✅✅ 强烈推荐（A类作品主场；国家级一等奖是升学硬通货） | ✅ 推荐（AI编程/发明作品仍可报；偏工程化和算法深度） | ❌ 不在参赛范围 | **青创赛 > 宋庆龄 > 雏鹰杯** |

### 4.2 各比赛「年龄侧重」的核心差异

#### 青创赛：最「学术化」

| 学段 | 适配度 | 关键约束 | 重点考察 |
|---|---|---|---|
| 小学 | ⭐⭐ | 仅限B类；A类按中学生标准评审 | 选题与年龄匹配度、探究方法、收集证据能力 |
| 初中 | ⭐⭐⭐⭐ | B类为主；A类开始开放 | 科学方法规范、立论严谨性、调研深度 |
| 高中 | ⭐⭐⭐⭐⭐ | A/B类均可；A类获奖含金量最高 | 基础理论扎实、科研潜质、独立创新 |

**产品话术建议**：「青创赛是'科研训练场'，越往高年级优势越大。小学高年级可以冲B类拿证书，初中是黄金期，高中是一等奖主战场。」

#### 宋庆龄发明奖：最「全民化」

| 学段 | 适配度 | 关键约束 | 重点考察 |
|---|---|---|---|
| 小学 | ⭐⭐⭐⭐⭐ | 4类均可；**科技绘画类仅限小学** | 生活问题+动手做实物+完整表达创意 |
| 初中 | ⭐⭐⭐⭐⭐ | 发明/创意/AI编程；绘画类名额极少 | 从问题到实物的完整闭环；查新+附件完整 |
| 高中 | ⭐⭐⭐⭐ | 发明/AI编程/创意；绘画不参与 | 算法原理+数据处理逻辑+方法论 |

**产品话术建议**：「宋庆龄是'发明创造展'，4-6 年级和初中阶段是主力军，小学阶段有专属的科技绘画类。家长可以理解为'适合全年龄段，但小学阶段最友好'。」

#### 雏鹰杯：最「本地化」+「院士标准」

| 学段 | 适配度 | 关键约束 | 重点考察 |
|---|---|---|---|
| 小学中低年级 | ⭐⭐ | 仅「创新创意」类门槛较低 | 想象力+表达能力 |
| 小学高年级 | ⭐⭐⭐⭐ | 7大类均可报 | 过程完整、动手能力、答辩清晰 |
| 初中 | ⭐⭐⭐⭐⭐ | 核心组别；研究链条需完整 | 院士级「科学方法+真实问题」拷问 |
| 高中 | ❌ | **明确不在参赛范围** | — |

**产品话术建议**：「雏鹰杯是上海专属的'小院士'评选，初中阶段是核心组别，院士级评委意味着评审标准更'硬核'。小学高年级可以冲「小院士」，初中是主力战场。」

### 4.4 给 kechuang-mentor 产品的「年龄段路由建议」

#### 路由决策表

```python
def recommend_competition(age, city, has_lab, has_patent, research_depth):
    """基于用户画像推荐比赛"""

    # 1. 上海地区：雏鹰杯优先（本地含金量高）
    if city == "上海" and age in ["小学高年级", "初中"]:
        return {"primary": "雏鹰杯", "secondary": ["宋庆龄", "青创赛"]}

    # 2. 小学中低年级：宋庆龄唯一优选
    if age == "小学中低年级":
        return {"primary": "宋庆龄（科技绘画/创意）", "secondary": []}

    # 3. 小学高年级：3 个比赛都开放
    if age == "小学高年级":
        return {"primary": "宋庆龄", "secondary": ["青创赛B类", "雏鹰杯（如上海）"]}

    # 4. 初中：黄金期，全面参与
    if age == "初中":
        if has_lab or research_depth == "深":
            return {"primary": "青创赛（A类冲省赛）", "secondary": ["宋庆龄", "雏鹰杯（如上海）"]}
        else:
            return {"primary": "宋庆龄", "secondary": ["雏鹰杯（如上海）", "青创赛（B类）"]}

    # 5. 高中：青创赛主场
    if age == "高中":
        return {"primary": "青创赛", "secondary": ["宋庆龄"]}

    return {"primary": "宋庆龄", "secondary": ["雏鹰杯", "青创赛"]}
```

## 五、初中生（12-15 岁）适配度分析

**结论**：3 个比赛对初中生都开放，但**宋庆龄发明奖**和**雏鹰杯**对初中生最友好，**青创赛**需要更高的科研素养和规范性。

| 比赛 | 初中生适配度 | 关键考虑 |
|---|---|---|
| **青创赛** | ⭐⭐⭐ 中等 | 初中生可申报 B 类（日常生活小发明），但需注意：① 必须先经省赛获奖 ② 查重率≤30% ③ 现场终评含科研潜质测评（量表），对科研素养要求高 |
| **宋庆龄** | ⭐⭐⭐⭐⭐ 高 | 初中阶段重点考察「应用 AI 技术解决真实问题」+「数据处理逻辑」，对动手能力、查新规范、附件完整度要求具体；明确「生活问题+创新创造+动手实践」 |
| **雏鹰杯** | ⭐⭐⭐⭐⭐ 高 | 初中组是核心组别；强调「以科学方法解决真实问题」「从观察到实践、从创意到应用的完整科创过程」；过程孵化机制（专家培训+辅导）完善 |

## 六、3 个比赛的「获奖共性规律」（winning patterns 跨比赛汇总）

通过对近 3 年（2022-2025）获奖项目分析，提炼出 **7 大跨比赛获奖规律**：

1. **「以小见大」选题法**——3 个比赛一等奖的共同特点。从生活小痛点切入（防夹手、防淋雨、防呛咳），但解决方案要具体且经过实测。
2. **「社会热点+本地特色」**——结合地方文化/特产/环保议题（如平潭蓝眼泪、青海生态、上海非遗），增强独特性。
3. **「特殊人群关怀」**——聚焦孕妇、老人、儿童等弱势群体（孕妇环境监测仪、老年智能枕头、家庭膏方糖浆智能小帮手、老年人智能服药助手），触发评委同理心。
4. **「AI+传统场景」**——用人工智能赋能日常生活（智能垃圾分类、教室通风系统、智慧龙影皮影戏），在青创赛一等奖中占比已达 55%+。
5. **「有专利或专业认证」**——发明专利>实用新型>论文发表，能佐证创新性和研究深度。
6. **「可验证的真实数据」**——用对照实验、问卷调查、热力图等呈现量化结果，比定性描述更有说服力。
7. **「学生全程参与+附件可追溯」**——从发现痛点→调研→设计→制作→测试→迭代，每一步都有学生记录与反思，且附件（图纸、照片、原始数据）齐全可追溯。

## 七、学生最常犯的 8 大错误（common pitfalls 跨比赛汇总）

1. **选题过大或太空泛**（如「如何保护环境」「人工智能改变生活」），缺乏具体可执行的切入点。
2. **学生主体性不足**——研究工作主要由教师/家长代做，答辩时被评委识破直接淘汰。
3. **创新性缺乏**——选题常见（「智能xxx」泛滥），方法为常规传感器/单片机组合，无新意。
4. **数据造假或润色过度**——青创赛报告查重率超 30% 即取消资格，原始记录缺失被判违规。
5. **研究过程不连续**——临时拼凑作品，从选题到完成只用了几个星期，完整性不足。
6. **类别选错**——小学生申报 A 类作品将按中学生标准评审；纯软件报入「创造发明」类；论文类报宋庆龄（明确不接受）。
7. **展板/材料出现禁忌信息**——含指导教师姓名、专家评价、媒体报道、过往获奖、专利申请情况等（青创赛明确禁止）。
8. **作品仅停留在创意设计阶段**——宋庆龄发明作品和雏鹰杯创造发明类必须有实物或仿真模型，纯概念图直接淘汰。

## 八、产品设计的关键洞察（≤500 字）

**给 kechuang-mentor 产品的 10 大设计建议（含年龄维度）**：

1. **「三档用户+地区」路由**：登录后先问「年级+城市+是否上海」，**直接推荐 1-2 个比赛**（详见第四章路由决策表）。
2. **「以小见大」选题教练**：内置 50+ 「生活小痛点」案例库，引导学生从身边场景切入，而非「保护地球」「AI 改变世界」这种伪命题。**小学阶段增加「科技绘画/创新创意」专属引导**。
3. **「真实用户调研」强制工作流**：要求学生至少访谈 3 位目标用户、收集 ≥30 份问卷，作为可行性验证的硬性门槛（这是 3 个比赛的共性考察点）。**小学阶段可降低到 1 位用户+10 份问卷**。
4. **「学生主体性」自检清单**：每完成一个阶段，反问「这件事能从头到尾讲清楚吗？」「如果评委问某个细节能回答吗？」，避免答辩时露馅。**小学阶段用「给爸妈讲一遍」替代**。
5. **「类别选择向导」**：根据学生想法推荐申报类别——A类（专业研究）vs B类（生活发明）vs 发明/创意/AI/绘画（宋庆龄）vs 7 大方向（雏鹰杯）。**小学用户优先推荐「绘画/创意/小发明」类**。
6. **「创新性 + 实用性」双轨评分卡**：3 个比赛权重最高的两个维度应作为产品的「快速自评」模块，用 5 分制 + 评分标准给学生即时反馈。**评分标准按学段动态切换**（用 `scoring_by_age` 字段）。
7. **「AI 赋能传统场景」加分项**：调研发现 AI 主题占青创赛一等奖 55%+，应主动引导学生思考「能不能用 AI/传感器/编程改造一个传统场景」（如皮影、垃圾分类、教室通风）。**小学阶段重点推荐「AI体验型」而非「算法型」**。
8. **「附件材料清单」自动生成**：根据申报类别+学段自动生成所需附件清单（图纸、照片、查新报告、原始数据、演示视频≤1分钟/3分钟），避免临阵发现缺材料。**小学低年级附件简化（如不要查新报告）**。
9. **「风险红线」弹窗提示**：内置绝对禁区清单（食品/医药/论文/脊椎动物/毒害物质/查重率>30%），学生输入选题时自动检测，提前规避。**高中阶段加强学术规范红线**。
10. **「长期培养路径规划」**：基于学生当前学段，生成 1-3 年的参赛路线图（参考第四章 4.3 节），避免临时抱佛脚。

**给开发 Agent 的路由建议**：

- 评估路由应支持「按比赛」「按学段」「按地区」「按作品形态」**四维筛选**。
- 每个比赛应输出结构化 JSON（已提供），含 `evaluation_dimensions` / `age_group_focus` / `winning_patterns` / `common_pitfalls` / `recommended_topics`，可直接被产品作为「评估 checklist」渲染。
- 建议在产品中增加「**匹配度测试**」功能：学生输入年龄+城市+作品想法，自动推荐最适合的 1-2 个比赛。
- 强烈建议增加「**评审模拟器**」功能：基于本 JSON 的评分标准，AI 扮演评委提问，验证学生主体性。
- 建议在产品中增加「**长期路线图**」功能：基于学段生成未来 1-3 年参赛路径，标注每个阶段的能力积累目标。
